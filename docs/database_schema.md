# Database Schema

AstroLyrica uses YAML files in `data/` as the initial symbolic database.

## Core collections

- `planets`: planetary meanings, images, verbs, body language, shadow language, medicine language, and avoid lists.
- `signs`: zodiacal element, modality, temperament, body zone, images, verbs, gifts, risks, and avoid lists.
- `houses`: traditional topics, modern topics, poetic locations, verbs, shadow language, and medicine language.
- `aspects`: angle, poetic logic, sentence patterns, constructive uses, and avoid lists.
- `conditions`: technical meaning, images, tone, interpretive rules, and avoid lists.
- `voices`: generation voice presets.
- `forms`: output form definitions and constraints.
- `banned_phrases`: phrases that should be rejected or rewritten.
- `source_notes`: bibliography and source provenance notes.
