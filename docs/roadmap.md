# Roadmap

## Phase 1: Repository foundation

- Create data folders, documentation, prompt templates, and validation tooling.
- Seed the symbolic database with starter entries.

## Phase 2: Local prototype

- Load YAML data.
- Accept manual astrology input.
- Retrieve relevant symbolic entries.
- Build generation prompts.
- Save generated prompt packages and draft outputs.

## Phase 3: Editorial workflow

- Add review metadata, ratings, and revision notes.
- Begin Scriptorium dashboard planning.

## Phase 4: Generation integrations

- Add optional LLM API calls after prompt packages are stable.
- Add quality checks for banned phrases and source compliance.

## Phase 5: Publishing automation

- Add Starweather scheduling and social publishing only after review workflows are reliable.
