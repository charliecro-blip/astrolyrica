# AstroLyrica Product Spec

AstroLyrica turns structured astrology symbolism into original poetic and practical outputs.

## Goals

- Maintain a curated symbolic language database.
- Generate meaning briefs, poems, horoscopes, and image prompts from manual astrology inputs.
- Save outputs for experiment tracking and editorial review.
- Prepare foundations for later dashboards and publishing workflows.

## Non-goals for initial setup

- No social posting.
- No LLM API calls.
- No astrology chart calculations.
